**关键点**

### 一. vscode

1.  VS Code 新增一个“粘性滚动”的设置：editor.experimental.stickyScroll.enabled
   
   当前代码块的第一行，可以始终保持在编辑器顶部

2.  快捷键折叠所有 ctrl + k + 0
   
    快捷键打开所有 ctrl + k + j

3. 运行 npm script run 插件 (NPM-scripts)
装完之后, 一开始插件默认是 auto, 在 vscode setting 中搜索 Package Manager 进行设置
